## ⚙️ Personnalisation

### Modifier les Couleurs
```css
/* Dans style.css - Variables CSS */
:root {
    --primary-color: #votre-couleur;
    --accent-color: #votre-accent;
}
```

### Adapter le Contenu
1. **Textes** - Modifier dans `index.html`
2. **Compétences** - Ajuster les pourcentages dans les barres de progression
3. **Projets** - Remplacer les exemples par vos réalisations
4. **Contact** - Mettre vos vraies informations

### Animations
```javascript
// Dans app.js - Personnaliser les durées
element.style.transition = 'all 0.8s cubic-bezier(0.4, 0, 0.2, 1)';
```
